# Sistema de Ventas - Java Swing

## Descripción
Aplicación de escritorio desarrollada en Java utilizando Swing para la gestión de ventas.  
Permite registrar ventas, calcular totales y ganancias, y mostrar la información en una tabla dinámica conectada a una base de datos.

Este proyecto forma parte de mi proceso de formación como desarrollador Java.

---

## Funcionalidades

- Registro de ventas
- Cálculo automático de total
- Cálculo de ganancia
- Conexión a base de datos MySQL
- Visualización de datos en JTable
- Arquitectura organizada por capas (DAO, Modelo, Vista)

---

## Tecnologías utilizadas

- Java
- Java Swing
- MySQL
- JDBC
- Git
- GitHub

## Cómo ejecutar

1. Clonar el repositorio
2. Abrir el proyecto en NetBeans
3. Configurar la base de datos en `ConexionBD.java`
4. Ejecutar la clase `main`
